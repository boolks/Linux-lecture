FIle 입출력
FILE fp - 구조체 변수 선언
fopen("경로", "모드") - 파일 열기
fgets(문자열, size, 파일) - 파일 불러와서 읽기
fputs("문자열", 파일) - 파일에 문자열 써넣기
fclose(파일) - 파일 닫기

옵션
"r" - 읽기 모드 (read)
"w" - 쓰기 모드 (write)
"a" - 붙여넣기 모드 (append)
