제어문
반복문 : while, for, do-while
조건문 : if, switch, continue, break, 다중 if

for문 - 반복횟수가 정해져 있을 때
while문 - 주로 긍정문에 사용 (~하는 동안)
do-while문 - 주로 부정문에 사용 (~하지 않는 동안)
조건이 처음부터 거짓이면 : for, while은 한 번도 실행 안함
                         do-while은 일단 한 번은

FIle 입출력
FILE fp - 구조체 변수 선언
fopen("경로", "모드") - 파일 열기
fgets(문자열, size, 파일) - 파일 불러와서 읽기
fputs("문자열", 파일) - 파일에 문자열 써넣기
fclose(파일) - 파일 닫기

옵션
"r" - 읽기 모드 (read)
"w" - 쓰기 모드 (write)
"a" - 붙여넣기 모드 (append)
